### SAMBA-MPath

Shard-level adaptive block structure for efficient verifiable query processing in BIoV environments.

SAMBA-MPath combines Hyperledger Fabric, IPFS, RSU local cache, and peer cache retrieval to reduce traffic-state payload query latency without replicating all payloads to every RSU. This project represents my research direction in distributed network environments, distributed storage, and verifiable data access.

Main contributions:

- Designed a shard-level adaptive block access structure for BIoV environments
- Implemented a Fabric/IPFS prototype for verifiable traffic-state data retrieval
- Developed RSU local, peer, and external payload retrieval paths
- Built workload replay, K/L trade-off analysis, and experiment summary generation

Achievements:

- Domestic conference paper accepted: 대한전자공학회 하계종합학술대회
- Korean patent application related to shard-based intelligent vehicle network systems
- Built a working prototype using Fabric as a control-plane and IPFS as distributed external storage

Tech stack: Python, Go chaincode, Hyperledger Fabric, IPFS/Kubo, Docker Compose, pandas, Matplotlib
